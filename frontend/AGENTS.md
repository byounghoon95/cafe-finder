# Frontend Agent Guide

## Stack

- React
- Vite
- TanStack Query
- Zustand
- Google Maps JavaScript API
- Tailwind CSS

## Standards

- Build the map app as the first screen.
- Keep controls compact and practical: radius selector, filters, sorting, result list, details.
- Show loading, empty, and error states.
- Avoid marketing-page layout for the main app.

## Verification

Run lint/tests when available and verify the map UI in browser after meaningful frontend changes.
